This project is an end-to-end Walmart Sales Data Analysis solution that demonstrates the complete analytics workflow, from raw data processing to business insights. Python was used for data cleaning, preprocessing, and exploratory data analysis, while SQL was used to perform advanced analytical queries and solve key business questions. The project highlights practical data analysis techniques, structured problem-solving, and the ability to transform raw data into actionable insights for business decision-making.
